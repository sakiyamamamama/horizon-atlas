6. 条件分岐（if文） (1)

python

if 条件式:
    #ここに、条件式が成立するときの処理を書く
else:
    #ここに、条件式が成立しなかったときの処理を書く

python

score = 75
print("あなたは", score, "点です")
if score >= 70:
    print("合格です")
else:
    print("不合格です")

text

＜出力結果＞
あなたは 75 点です
合格です

score = 75
print("あなたは", score, "点です")
if score >= 90:
    print("あなたの評価はSです")

    print("あなたの評価はAです")
elif score >= 70:
    print("あなたの評価はBです")
elif score >= 60:
    print("あなたの評価はCです")
else:
    print("あなたの評価はDです")

text

あなたは 75 点です
あなたの評価はBです
